Recommendation

Stabilize Women's before you copy patterns site-wide

Any pattern deployed from Women's will propagate to Men's and Jordan. Shipping unstable patterns multiplies the remediation cost.

recommendationlist

Fix CLS by reserving layout space for hero images before load | Add explicit width/height attributes and CSS aspect-ratio to prevent reflow | priority

Block any new pattern from Women's until CWV targets are met | Audit Women's for render-blocking resources before the next release | risk

3

Recommendation

Shrink the shared critical path across flagships

Hero image weight and tag load order are controllable today. Fixing them site-wide would lift every template score meaningfully.

recommendationlist

Set 200KB hero image budget enforced in CI | A single oversized hero image adds 2–3s to LCP on mobile — the budget prevents regression | priority

Move non-critical tags to post-interactive loading | Third-party tags account for 28% of delivery path delay and are safe to defer | action
